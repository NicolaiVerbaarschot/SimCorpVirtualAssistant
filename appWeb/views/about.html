About

The scope of this project is to investigate to which extend a virtual assistant (or chatbot) can be used to improve the user experience of a browser based asset management system. It is a collaboration between SimCorp and DTU and forms part of the DTU course "02122 - Fagprojekt" for software engineers. The project requirements were given by SimCorp which acts as the client in a software engineering context. The chatbot must be able to perform basic table operations such as sorting, filtering, grouping, etc. while communicating smoothly with a user through written or spoken natural language, thus providing a significant improvement to the traditional mouse and keyboard based interaction. In addition to the basic features, the system is also able to visualise data and has a superuser mode that provides a terminal-like interaction for experienced users which is based on commands rather than natural language.
